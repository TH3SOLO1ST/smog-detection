## ðŸŽ¬ Production Guidelines

### Video Quality Standards

**Technical Specifications:**
- Resolution: 1920x1080 (Full HD)
- Frame rate: 30 fps
- Audio: Clear voiceover, background music optional
- Format: MP4 for web distribution
- Length: As specified per script

**Visual Style:**
- Clean, professional appearance
- Consistent branding throughout
- Clear, readable text overlays
- Smooth transitions and animations
- Highlight important elements

### Content Guidelines

**Language and Tone:**
- Simple, non-technical language
- Friendly, encouraging tone
- Clear step-by-step instructions
- Avoid jargon and technical terms
- Speak slowly and clearly

**Accessibility:**
- Closed captions for all videos
- High contrast visuals
- Clear, large text
- Descriptive audio for key actions
- Multi-language support (future)

### Recording Guidelines

**Screen Recording:**
- Use clean desktop environment
- Hide personal files and bookmarks
- Use consistent browser settings
- Ensure all elements are visible
- Test recording quality before final

**Voice Recording:**
- Use quality microphone
- Record in quiet environment
- Maintain consistent volume
- Edit out mistakes and pauses
- Add background music if desired

### Post-Production

**Editing:**
- Remove mistakes and long pauses
- Add zoom effects for important areas
- Include text overlays for key points
- Add transitions between sections
- Ensure smooth flow

**Quality Assurance:**
- Test all instructions work as shown
- Verify all text is readable
- Check audio quality
- Ensure video meets length requirements
- Get feedback from test users
